内置组件说明

食用方法：将ext.lua放入工程文件夹下，引用方法如下

require "import"
import "ext"

//远程更新数据功能,服务端格式要求请查看(Service.md)

若使用远程功能请在程序启动里面加入微云链接

local ServiceUrl="http://share.weiyun.com/"

1--远程更新及公告

远程更新()
远程公告()

注：远程更新(true)
可附带参数，false/true，功能为是否显示(正在检查更新......)悬浮窗

2--加载远程链接

远程链接("url01")

//本地扩展功能组件说明

设备信息()

屏幕切换()
